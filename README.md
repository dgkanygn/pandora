> 🌐 **Canlı Yayın Adresi:** Bu proje freelance bir proje olup aktif olarak **[pandoracicek.com](https://pandoracicek.com/)** adresinde yayındadır. Şu anda sistem üzerinde bazı ufak entegrasyon çalışmaları ve geliştirmeler sürdürülmektedir.
 
React/TypeScript ve PHP + MySQL kullanılarak geliştirilen bu full-stack projede, işletme için modern bir web sitesi ve tüm dinamik içeriklerin (kategoriler, ürünler, siparişler vb.) teknik bilgi gerektirmeksizin yönetilebildiği kapsamlı bir admin paneli (CMS) inşa edilmiştir.

---

## 🛠️ Tech Stack

* **Frontend:** React/TypeScript
* **Admin Panel:** React/TypeScript
* **Backend / API:** PHP
* **Veritabanı:** MySQL
